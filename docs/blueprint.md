# **App Name**: StockPilot

## Core Features:

- User Authentication: Secure login/logout system with role-based access control (Admin, Staff).
- Product Management: Add, edit, and categorize products with details like name, SKU, price, and image upload.
- Real-time Stock Tracking: Automatically update inventory levels upon each transaction.
- Low Stock Notifications: Configure and receive alerts when stock levels fall below defined thresholds.
- Transaction Management: Record stock in/out events and generate sales orders with printable invoices.
- Inventory Forecasting Tool: AI-powered tool that analyzes sales data and predicts future demand to help optimize stock levels. It considers seasonal trends and suggests reorder quantities. Note: Requires integration of historical sales data.
- Dashboard and Reporting: Visualize key metrics like total stock value, sales summaries, and profit/loss reports.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) for trustworthiness and stability.
- Background color: Light Grey (#F5F5F5), a desaturated version of the primary, for a clean backdrop.
- Accent color: Purple (#7E57C2), an analogous color to the primary, for highlighting key actions.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral user experience.
- Simple, clear icons for navigation and actions within the inventory system.
- Responsive design to ensure seamless usage on desktop, tablet, and mobile devices.
- Subtle transitions and animations for a smooth user experience.
